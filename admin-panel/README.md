## Admin panel codes should be inside this directory# Waste Management System - Admin Dashboard

A centralized web platform used by city officials to monitor live operations and manage system resources.

## 📁 Directory Structure
- `src/components/`: Reusable UI elements (Charts, Tables, Map views).
- `src/pages/`: Admin Dashboard, Analytics, User Management, and Route Assignment.
- `src/services/`: API calls to the backend and live data fetching.
- `src/hooks/`: Custom React hooks for global state management.

## 🚀 Key Features Implemented
- **Live Operation Monitoring**: Real-time tracking of trucks and schedules.
- **Route Assignment**: Tools to add/edit routes and assign trucks to specific batches.
- **Analytics & Reporting**: Generates statistics on recycling efficiency and operational delays.
- **Data Access**: Exclusive access to archived system logs and deleted account info.

## 🛠 Tech Stack
- **Library**: React.js
- **Styling**: Tailwind CSS / Material UI
- **Maps**: Google Maps API (Live Vehicle Tracking)
